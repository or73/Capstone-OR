HOTEL MEAN - Main Page

Welcome to this MEAN Application.

Main objective

Use MEAN stack.

All users can access the site, and see details of a selected hotel, but only registered users can add reviews.

Features

  • User validation, with token.
  • User registration.
  • Select a hotel to see its details.
  • Add a review to a selected hotel, when user has been validated.
  • All information, of hotels and users, is stored in mongodb.
  • Use of:
    • Templates.
    • Factories.
    • Controllers.
    • Bootstrap, jquery.

Database

MongoDB

Packages

  • angular: is a development platform for building mobile and desktop web applications.
  • angular-route: allows to user $route, and it is used for deep-linking URLs to controllers and views (HTML partials). It watches $location.url() and tries to map the path to an existing route definition. ngView is a directive that complements the $route service by including the rendered template of the current route into the main layout (index.html) file. Every time the current route changes, the included view changes with it according to the configuration of the $route service.
  • body-parser: It returns a function that acts as middleware. The function listens for req.on(‘data’) and constructs req.body from the chunks of data it gets.
  • bootstrap: most popular front-end framework for developing responsive.
  • cordova:Apache Cordova allows for building native mobile applications using HTML, CSS and JavaScript. This tool helps with management of multi-platform Cordova applications as well as Cordova plugin integration.
  • exec-plan: Provide the ability to run child process commands sequentially, with some fine-grained control, all while avoiding the dread pyramid of doom (i.e., callback indentation).
  • express: Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.
  • ionic: is a complete open-source SDK for hybrid mobile app development. Built on top of AngularJS and Apache Cordova, Ionic provides tools and services for developing hybrid mobile apps using Web technologies like CSS, HTML5, and Sass.
  • jwt:JSON Web Token (JWT) is an open standard (RFC 7519) that defines a compact and self-contained way for securely transmitting information between parties as a JSON object. This information can be verified and trusted because it is digitally signed. JWTs can be signed using a secret (with the HMAC algorithm) or a public/private key pair using RSA.
  • mocha: Mocha is a feature-rich JavaScript test framework running on Node.js and in the browser, making asynchronous testing simple and fun. Mocha tests run serially, allowing for flexible and accurate reporting, while mapping uncaught exceptions to the correct test cases.
  • mongoose: is a Node.js library that provides MongoDB object mapping similar to ORM with a familiar interface within Node.js.
  • path-exists: checking if a file exists before opening.
